Expression: A fragment of code that produces a value is called an Expression.
Every value written is literally an Expression.

eg: 1809 or "Nikhil".

Operators:-

1 - Arithmetic Operators:-

+ -> Addition
- -> Subtraction
* -> Multiplication
** -> Exponential
/ -> Division
% -> Modulus
++ -> Increment
-- -> Decrement

2 - Assignment Operators

= -> x=y
+= -> x=x+y
-= -> x=x-y
*= -> x=x*y
/= -> x=x/y
%= -> x=x%y
**= -> x=x**y
