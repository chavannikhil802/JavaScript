Strings

Strings are used to store and manipulate text.
Strings can be created using the following syntax:-

let name = "Nikhil"

name.length => This property prints the length of the string

---------------------------------------------------------------------------------

Strings can also be created using single quote:-

let name = 'Nikhil'

---------------------------------------------------------------------------------

Template Literals:-

String can also be created using back tick;

eg: let name = `Nikhil`;

With Template Literal, we can use both single quote and double quotes.

Using ${var-name} in Template Literal is used to print the variable value.

it is called as Interpolation.

---------------------------------------------------------------------------------
